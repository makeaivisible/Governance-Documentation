# Dataset Release Policy

Draft for expert review.

## Release Requirements

- Only anonymized and approved records may enter a release pipeline.
- Public datasets should avoid raw transcripts unless specifically approved through a future policy.
- Cohorts below minimum thresholds must be suppressed or aggregated.
- Each release must include a manifest, schema, changelog, and dataset card.

## Fail-Closed Rule

If approval, schema validation, or privacy checks are missing, the release should fail.
