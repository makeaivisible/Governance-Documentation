# Human Review Protocol

Draft for expert review.

## Reviewer Access

Reviewers should only access anonymized records. Raw submissions should not be available in the review queue.

## Reviewer Tasks

- Confirm anonymization quality.
- Flag suspected remaining identifiers.
- Calibrate behavioral labels.
- Escalate privacy risks before scoring or publication.

## Disagreements

Disagreements should be tracked, discussed, and resolved through a documented calibration process.
