# Governance Documentation

Governance, methodology, ethics, and coordination docs for Make AI Visible.

This repository is the source of truth for how the project makes decisions, protects contributors, validates methods, and publishes research.

## MVP Scope

- Consent and assent draft language.
- Data lifecycle and retention policy.
- Privacy and redaction methodology.
- Human review protocol.
- Scoring validation plan.
- Dataset release policy.
- Contributor governance and community guidelines.

## Human Review Required

Several documents in this repository can be drafted in public, but they should not be treated as final legal, IRB, clinical, or security advice until reviewed by qualified people.

## First Milestone

Publish draft governance documents that let volunteers understand the project, open issues against unresolved decisions, and begin expert review.
